kili kipisi

My personal blog exclusively in Toki Pona, a minimalist constructed language. o tawa li lukin e ona a! The name is nonsensical, translating roughly to "sliced produce". It's pronounced like "KEE-lee KEE-pee-see".

TzStamp

An open source trusted timestamping toolkit built on the Tezos blockchain. Collaboration with, and currently maintained by JD Pressman.

Deno Roulette

A satirical website poking fun at developers who disregard the Deno runtime's security model and haphazardly run unaudited programs like they would with Node.js.

Illager Net

Hardcore Minecraft server. Randomized spawn and a 3-day ban upon death. The server has been running since August 2019 without reset.

Journey Under The Sea

Proof-of-concept JavaScript-free adaptation of the Choose Your Own Adventure book by R. A. Montgomery. This site is distributed through the peer-to-peer Hypercore protocol, rather than hosted on a centralized server. You'll need to download a special browser like Beaker to access it.
